Event Pages Update Guide

To update all event pages with the unified FAQ accordion functionality, follow these steps:

Step 1: Add necessary files to each event page

Add these lines right before the closing </body> tag:

<script src="../js/faq-accordion.js"></script>
<link rel="stylesheet" href="../css/faq-accordion.css">

Step 2: Remove existing FAQ script

Remove the existing FAQ accordion script which typically looks like this:

<script>
    // FAQ Accordion functionality
    document.addEventListener('DOMContentLoaded', function() {
        const faqQuestions = document.querySelectorAll('.faq-question');
        
        faqQuestions.forEach(question => {
            question.addEventListener('click', function() {
                this.classList.toggle('active');
                
                const answer = this.nextElementSibling;
                if (this.classList.contains('active')) {
                    answer.style.display = 'block';
                } else {
                    answer.style.display = 'none';
                }
            });
        });
    });
</script>

Step 3: Verify FAQ HTML structure

Make sure your FAQ HTML follows this structure:

<div class="faq-section">
    <div class="faq-item">
        <div class="faq-question">How do I register for the event?</div>
        <div class="faq-answer">
            Registration details here...
        </div>
    </div>
    
    <div class="faq-item">
        <div class="faq-question">What are the rules?</div>
        <div class="faq-answer">
            Rules details here...
        </div>
    </div>
</div>
